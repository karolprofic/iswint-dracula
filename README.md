# ISWinT Dracula

**ISWinT Dracula** is a 2D platform game in which you play as Dracula, who must travel to ISWinT.

---

## Controls

| Key            | Action                   |
|----------------|--------------------------|
| `A`, `D`       | Move character           |
| `Space`        | Jump / Special action    |
| `1`            | Healing potion           |
| `2`            | Umbrella                 |
